DIAGRAM

The DIAGRAM content model was developed to standardize the nature of these resources.

The following l overview of the primary content elements.

description
The description element is the root element of each DIAGRAM description, and contains an optional head and required body element
head
The head element contains any metadata associated with the described resource, but none is currently required
body
The body element defines the various descriptions and image types associated with the described resource
summary
The summary element contains a succinct version of the description
longdesc
The longdesc element contains a comprehensive explanation of the referenced resource
simplifiedLanguageDescription
The simplifiedLanguageDescription element contains a simplified language version of the description
tactile
The tactile element contains an alternative tactile image
simplifiedImage
The simplifiedImage element contains a simplified version of the source image
